
# Harmonicabsorber Wed Feb 10 2021 14:31:51 GMT+0100 (Central European Standard Time)

Re-analysis of data from report 16. Compare to report 23.
This report increases the confidence level to three sigma,
uses a 15% cutoff on both sides for the m estimator and increases
the number of rounds for the m estimator from 15 to 35. Almost
no difference is visible. Confidence intervals are slightly larger;
graphs are slightly smoother, but that might be more confirmation bias.
